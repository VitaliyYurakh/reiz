## Сайт — Можливості

### Мови та валюти
- **5 мов**: Українська (основна), Російська, Англійська, Польська, Румунська
- **3 валюти**: UAH / USD / EUR — конвертація в реальному часі з 12-годинним кешем
- SEO-оптимізація: Schema.org розмітка, мета-теги, sitemap для кожної мови

### Сторінки

| Сторінка | Опис |
|---|---|
| **Головна** | Hero-слайдер, каталог з фільтрами, переваги, як орендувати, FAQ |
| **Каталог** `/cars` | Всі авто з фільтрацією за сегментом, ціною, характеристиками |
| **Авто** `/cars/[id]` | Фото (мобільні/PC), характеристики, тарифні плани, бронювання |
| **Про нас** `/about` | Місія, історія, статистика, команда |
| **Бізнес** `/business` | Корпоративна оренда з ПДВ, калькулятор, кейси |
| **Інвестиції** `/invest` | Інвестиційні можливості, ROI-калькулятор, порівняння |
| **Страхування** `/insurance` | Пакети покриття (Standard / Comfort / Premium) |
| **Блог** `/blog` | Статті та тревел-гайди |
| **Сертифікат** `/certificate` | Подарунковий сертифікат |
| **Контакти** `/contacts` | Офіси на карті, форми зворотного зв'язку |
| **FAQ** `/faq` | Часті запитання з розкривними блоками |
| **Умови** `/terms` | Юридичні умови та договори |

### Бронювання (multi-step модалка)
1. Вибір дат та локації, тарифного плану, додаткових послуг
2. Контактна інформація (ім'я, телефон, email, рейс)
3. Підсумок цін з розбивкою
4. Підтвердження бронювання

### Ключові фічі сайту
- Адаптивний дизайн (мобільний + десктоп)
- Оптимізація зображень (Next.js Image, MOBILE/PC варіанти фото)
- ISR (Incremental Static Regeneration) для швидкості
- Turbopack для розробки
- Анімації при скролі (AOS)
- Структуровані дані (JSON-LD)
- Міжнародний телефонний ввід з кодами країн

## Адмін-панель — Можливості

### Загальне
- Темна / Світла тема з перемикачем
- 3 мови інтерфейсу: UK / RU / EN
- RBAC — рольовий доступ (Admin / Manager / Staff)
- Аудит-логування всіх дій
- Skeleton-завантаження на кожній сторінці

### Модулі

#### 1. Dashboard
- KPI-картки: активні оренди, місячний дохід, утилізація автопарку, прострочені оренди
- Графік доходу (Recharts)
- Графік утилізації
- Алерти прострочених оренд

#### 2. Заявки (Requests)
- Список з пагінацією, пошуком, фільтрацією за статусом
- Статуси: Нова → В обробці → Схвалена / Відхилена / Скасована
- Джерела: website, phone, telegram, instagram, walk-in, referral
- Створення заявки вручну
- Схвалення заявки → автоматичне створення бронювання

#### 3. Бронювання (Reservations)
- Статуси: Підтверджено → Видано → Скасовано / Неявка
- Пакет покриття та додаткові послуги
- Перехід до оренди

#### 4. Оренди (Rentals)
- Статуси: Активна → Завершена / Скасована
- Номер контракту
- Огляд авто (інспекції з фото)
- Штрафи та пошкодження
- Продовження оренди
- Генерація документів

#### 5. Календар
- Горизонтальна timeline-сітка (14–56 днів)
- Лінія "зараз" з автооновленням
- Фільтрація за авто
- Перевірка доступності на обрані дати
- Кольорове кодування: оренди (синій), бронювання (бірюзовий), сервіс (помаранчевий)
- Клік по події → деталі + посилання

#### 6. Клієнти (Clients)
- Профілі з рейтингом, документами, історією оренд
- Пошук за ім'ям, телефоном, email
- Аватар з ініціалами (градієнтний фон)
- Джерело клієнта (website / walk-in / referral / phone)
- Блокування клієнтів з причиною

#### 7. Автопарк (Cars)
- Grid-картки з фото, ціною, статусом
- Статус доступності (зелений / червоний)
- Бейджі: NEW, знижка %
- Сегменти (economy / comfort / premium)
- Редагування: фото, тарифи, характеристики

#### 8. Локації (Locations)
- Міста з мультимовними назвами (uk / ru / en)
- Пункти видачі / повернення
- Статус "Популярне" та "Активне"

#### 9. Ціноутворення (Pricing)
- **Тарифні плани** — ціни за день, мінімальний строк, сезонність
- **Додаткові послуги** — GPS, дитяче крісло, додатковий водій
- **Пакети покриття** — Standard / Comfort / Premium з матрицею покриття

#### 10. Фінанси (Finance)
- Рахунки (каса, картка, банк) з балансами в реальному часі
- Прихід / Витрата / Баланс на кожній картці
- Клік по картці → фільтрація транзакцій
- Inline-редагування рахунку
- Транзакції: 8 типів (оплата, депозит, повернення, штраф, сервіс, ...)
- Прив'язка до клієнта / оренди / бронювання
- Мультивалютність з курсом конвертації
- Створення транзакції з формою

#### 11. Сервіс та обслуговування
- Типи: ТО, ремонт, страховка, ТО держ., мийка, інше
- Прив'язка до авто
- Блокування бронювань на час сервісу
- Вартість, виконавець, дати

#### 12. Звіти (Reports)
- KPI-дашборд з трендами
- Звіт по доходах за період
- Утилізація автопарку за період
- Прострочені оренди

#### 13. Налаштування (Settings)
- **Шаблони** — Email/SMS шаблони для сповіщень
- **Аудит** — Журнал дій з фільтрацією (хто, що, коли, before/after)
- **Команда** — Управління користувачами та ролями
- **Профіль** — Зміна пароля, налаштування акаунту

## Безпека

- JWT-аутентифікація з httpOnly cookies
- RBAC з гранулярними дозволами (13 модулів × full/view/none)
- Валідація всіх вхідних даних через Zod
- Rate limiting на API
- Helmet security headers
- CORS з white-list доменів
- CSRF-захист
- Аудит-логування всіх змін (хто, що, коли, IP, User-Agent)
- Хешування паролів (bcrypt)
- Валідація env-змінних при старті
- Graceful shutdown (SIGTERM/SIGINT)
- Request ID в кожному лог-записі (AsyncLocalStorage + Pino mixin)
